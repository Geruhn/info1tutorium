formatierung

“Formatiere und schreibe deinen Code, als würde ein wahnsinniger Psychopath, der weiß wo du schläfst, deinen Code reviewen.”

  • Schreibe nicht mehr als 80 Zeichen pro Zeile (zumindestens nie mehr als auf einmal auf dem Bildschirm zu sehen sind)
  • Setzte vor und nach jedem binären Operator ein Leerzeichen wie bei

  • Vergleiche Boolesche Werte nicht mit ==

    und statt esRegnet == false schreibe !esRegnet
  • Wähle für einen Ausdruck die selbe Operatorenreihenfolge wie in einem mathematischen Ausdruck, implementiere 0 < i < j < n als 0 < i && i < j && j < n
  • Vermeide Seiteneffekte

Variablen und Datentypen

Aufgabe

Die Klimaanlage im Hörsaal HE hat eine sehr komplizierte Steuerung. Sie hängt von den Folgenden Messwerten ab:

  • Anzahl Studenten im Saal, zum Beispiel 35
  • Anzahl offener Türen, zum Beispiel 1
  • Uhrzeit in Stunden (1-24), zum Beispiel 1 bei 1:12 Uhr oder 14 bei 14:59 Uhr
  • Außentemperatur in Grad Celsius, zum Beispiel 33,45 Grad Celsium
  • Licht an/aus, zum Beispiel false

Deklarieren Sie Variablen mit geeigneten Datentypen und den Beispielwerten als initialen Wert für jeden dieser Meßwerte.
Beachten sie dabei die Konventionen aus der Vorlesung.

Lösung

  • Anzahl Studenten im Saal, zum Beispiel 35
    • int numberOfStudents = 35;
  • Anzahl offener Türen, zum Beispiel 1
    • int openDoors = 1;
  • Uhrzeit in Stunden (1-24), zum Beispiel 1 bei 1:12 Uhr oder 14 bei 14:59 Uhr
    • int hour = 14;
  • Außentemperatur in Grad Celsius, zum Beispiel 33,45 Grad Celsium
    • double temperaturCelsius = 33.45;
  • Licht an/aus, zum Beispiel false
    • boolean lightOn = false;

Boolscher Ausdruck

boolean erdeDrehtSich = galileo && doch;

Aufgabe

Die Klimaanlage ist in Betrieb, falls:

  • zwischen 17 und 56 Studenten im Saal sind (einschließlich 17 und 56), nicht mehr als 2 Türen offen sind und die Uhrzeit in Stunden ungerade ist; oder
  • das Licht an ist und die Außentemperatur größer als 38,56 Grad Celsius ist.

Geben Sie diesen Sachverhalt als einen Boolschen Java-Ausdruck mit den aus der vorherigen Teilaufgabe deklarierten Variablen an. Beachten Sie die Formatierungsregeln für Ausdrücke.

Lösung


							boolean klimaanlageAn = 
								( 17 <= numberOfStudents 
									& numberOfStudents <= 56 
									& openDoors <= 2 
									& ( hour % 2 == 1 ) ) 
								| ( lightOn & temperaturCelsius > 38.56 );
